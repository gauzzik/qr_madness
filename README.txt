Just for fun
creates as much qr as you want from .txt file
prerequesites:
  pip install qrcode[pil]
Usage
  qr.py path_to_input_file path_to_generation_folder number_of_qr_to_generate

just like this

  qr.py input.txt qr/new_generation 5

this will generate 5 qr from input.txt an save it in qr/new_generation folder
